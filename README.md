Paradise Nursery is an online plant shop built with React and Redux Toolkit. Users can browse plants, add them to a cart, update quantities, and view total costs.

- Product listing with categories
- Redux-powered shopping cart
- Dynamic cart quantity & pricing

- Redux Toolkit
- React Router DOM
