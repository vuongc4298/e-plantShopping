e-plantShopping is a React-based shopping cart application developed for the **Paradise Nursery online plant shop**.  
The application allows users to browse different categories of house plants, add plants to a shopping cart, manage quantities, and view total costs dynamically.

This project was built as a final assignment to demonstrate skills in **React components, state management with Redux, and dynamic UI rendering**.

---

## Project Features

- Landing page with navigation to product listings
- Product listing page displaying multiple plant categories (e.g., Aromatic and Medicinal plants)
- Individual plant cards with image, description, price, and **Add to Cart** button
- Shopping cart page with:
  - Item thumbnail
  - Unit price
  - Quantity increase/decrease controls
  - Item total price
  - Remove item option
- Dynamic cart quantity and total cost updates using Redux
- Continue Shopping and Checkout buttons

---

## Technologies Used

- React
- Redux Toolkit
- React Router DOM
- JavaScript (ES6)
- CSS

---

## Learning Objectives

This project demonstrates:
- Creating reusable React functional components
- Managing global state using Redux Toolkit
- Handling user interactions and conditional rendering
- Dynamically rendering data using JavaScript array methods
- Structuring a small-scale front-end application
